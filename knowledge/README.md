## Maintenance rules

1. **Stability bar**: only settled knowledge goes here. Market views belong
   in intelligence/; rules belong in strategy/; ideas belong in research/.
2. **Additions** cite their source (book, paper, filing type, institution).
   No orphan claims. Contested topics present both sides.
3. **Learning loop**: when LESSONS_LEARNED or a post-mortem surfaces a
   knowledge gap, the fix lands here (and the lesson links to it).
4. **Review cadence**: annual review (playbooks/annual-review.md) includes a
   pass over this index — prune what proved useless, note what's missing.
5. These documents inform judgment; they are never cited as *evidence* for a
   specific trade (evidence must be current, per RESEARCH_PROCESS §5).
