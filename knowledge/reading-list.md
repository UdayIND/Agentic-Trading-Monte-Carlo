# READING LIST — Curated Sources With Purpose

Reference module. Each entry says what it's *for* — a source without a use
is decoration. Primary sources outrank commentary (working principle).
When a book/paper's idea gets absorbed into practice, note where.

## 3. Investor letters & memos (free, primary, ongoing)

- Berkshire Hathaway shareholder letters (Buffett, 1977–) — capital
  allocation, owner earnings, incentive reading.
- Howard Marks memos (Oaktree) — cycle and risk temperament, real-time.
- JPMorgan Guide to the Markets (quarterly) — the base-rate chartbook.
- Credit Suisse/UBS Global Investment Returns Yearbook (Dimson-Marsh-
  Staunton) — century-scale, survivorship-corrected return data.
- Aswath Damodaran's blog + data sets (NYU) — valuation practice and
  updated ERP/multiples data.
- AQR research library — factor evidence with methodology shown.

## 4. Primary data & filings (the desk's daily raw material)

- SEC EDGAR (filings, full-text search) — the research pipeline's substrate.
- FRED (St. Louis Fed) — every macro series in economic-data.md.
- BLS/BEA/Census/ISM release pages + calendars — the release map's sources.
- Federal Reserve: FOMC statements, minutes, SEP, speeches.
- Company IR pages: transcripts, decks, K/Q archives.

## 5. Usage discipline

1. New source → this list only with a stated purpose and tier
   (platform/TOOL_USAGE_GUIDE source tiers apply).
2. Ideas extracted from reading land as: knowledge-module updates,
   IMPROVEMENT_BACKLOG proposals, or research-template refinements —
   with the citation.
3. Nothing here is *evidence* for a live trade; evidence must be current
   and primary (RESEARCH_PROCESS §5).
